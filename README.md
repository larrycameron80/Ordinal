# Rune Swap Backend

## Overview

Rune Swap is a decentralized exchange (DEX) facilitating seamless token swaps on the Bitcoin network. This backend repository powers the core functionalities of the Rune Swap platform, enabling efficient and secure automated market-making (AMM) operations for Bitcoin-based assets.


## Features

- Automated Market Making (AMM): Enables liquidity pools for seamless token exchanges on the Bitcoin blockchain.​
- Token Swaps: Supports trading between various Bitcoin-native tokens, including Runes and Inscriptions.​
- Liquidity Provision: Allows users to add liquidity and earn rewards in Bitcoin-based tokens.​
- Secure Transactions: Implements robust security measures for safe and trustless trading.
